How to use this tool

Making changes without touching code.

The workflow
  1. Make changes in any panel
  2. Hit the Generate button
  3. Click Copy on the output
  4. Open the HTML file in a text editor (VS Code, TextEdit, Notepad)
  5. CSS snippets — find the matching CSS block and replace it
  6. Find & Replace pairs — use Cmd+H (Mac) / Ctrl+H (Windows)
  7. Save → push to GitHub → Vercel deploys in ~30 seconds
Tips
  • Use Save to browser so values persist between sessions
  • Keep admin.html in your portfolio folder locally — it never needs to go online
  • Colour preview updates in real time as you pick
  • Font previews load from Google Fonts — needs internet to preview
  • If something breaks, GitHub has your full history — you can revert any time
